# Requirements

## Project Objective
Create an MT5 Expert Advisor shell focused on trading discipline, capital protection, and structured data collection.

## Scope
- Provide a modular MQL5 architecture.
- Keep the initial release documentation-only and structural.
- Avoid strategy logic and automated trading behavior in this phase.

## Non-Goals for This Phase
- No indicator implementation.
- No auto-trading logic.
- No AI workflow integration.
